The List (updated 1/19/2026)

This is a WWW version of The Boston List's excellent (Boston) Greater Boston Area concert guide. All of the content is curated by local volunteers, we've just HTMLized everything. Please send us mail if you have questions or corrections related to the list content. My program automatically converts the listings into HTML, so please mail us if you'd like to add or update a show.

Here's what the symbols at the end of each listing mean: 

  *     recommendable shows                 a/a   all ages
  $     will probably sell out              @     pit warning
  ^     under 21 must buy drink tickets     #     no ins/outs
